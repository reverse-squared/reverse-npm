# @reverse/array
> Useful functions for managing arrays.

## Install
```
npm install @reverse/array
```

## Table of Contents
- [shuffle](#shufflearray)

--- 

## Usage
### shuffle(array)
> Shuffles an array and returns it.
#### Paramerters
- `array: Array`: The array to shuffle.
#### Example
```js
import { shuffle } from '@reverse/array';

shuffle([1, 2, 3]);
// Example Output: [2, 3, 1]
```
